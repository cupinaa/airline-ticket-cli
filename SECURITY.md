# Security Policy

## Scope

Airline Ticket CLI is an educational, local, CSV-backed application. It is not designed for production authentication, concurrent access, or storage of real passenger information.

## Safe usage

- Use only synthetic data.
- Do not reuse the bundled demonstration passwords elsewhere.
- Do not commit real passenger details or credentials.
- Do not expose the data directory through a public service.

## Reporting a vulnerability

Use GitHub's private vulnerability reporting feature when it is enabled for the repository. Do not include personal data or active credentials in a public issue.
